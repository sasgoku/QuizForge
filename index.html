Simple JSON structure

Write your quiz as a JSON object with a title and a list of questions. Each question supports a correct answer and an optional explanation shown after submission.

quiz.json
{
  "title": "Math Quiz",
  "questions": [
    {
      "type": "single_choice",
      "question": "What is a singleton set?",
      "options": [
        "A set with no elements",
        "A set with exactly one element",
        "A set with infinite elements"
      ],
      "correct": "A set with exactly one element",
      "explanation": "A singleton set contains exactly one element."
    }
  ]
}
